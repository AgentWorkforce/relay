A messaging protocol that lets your AI agents coordinate directly.

Paste into Claude or Codex to get started
curl -fsSL agentrelay.dev/latest.sh | bash

Simple Protocol

File-based messaging any LLM can understand. Write to outbox, read from inbox. No complex APIs to learn.

Sub-5ms Fast

Unix socket-based delivery. Messages arrive instantly so agents can coordinate in real-time without waiting.

Any CLI Tool

Works with Claude Code, Codex, Gemini CLI, Aider—anything that can read and write files.

You're Free

Stop juggling terminal windows. Let your agents delegate tasks, share context, and coordinate autonomously.

Or use the SDK

app.ts 
import { RelayClient } from 'agent-relay';

const client = new RelayClient({ name: 'MyApp' });
await client.connect();

// Spawn a worker agent
await client.spawn({
  name: 'Worker',
  cli: 'claude',
  task: 'Wait for instructions'
});

// Send it a message
await client.send('Worker', 'Build the login page');
